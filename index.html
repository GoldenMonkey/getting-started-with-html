The Web

The World Wide Web was invented in the early 1990's. The main type of document on the web is HTML and HTML is connected by hyperlinks. HTML

HTML stands for Hyper TextMarkup language.  It forms the heart of the web and is made up of text content and markup lanuage, references to other documents, and links to other pages.

Markup comsists of tags and elements. Tags are always contained in brackets and musy have opening and closing tags unless they are void.

HTML Attributes

Some tags contain attributes which assign a value that is applied to the element. An example would be the anchor tag which can be used to link to other pages like this.

Inline and Block Elements

Block elements differ from inline elements in that they form a box around their content.
